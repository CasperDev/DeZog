• NEW
    The 3.0 release of DeZog is dedicated to Reverse Engineering. A lot of improvements have gone into the internal memory model, the disassembler and the interactive way of disassembling unknown code.
  • NEW
    Disassembler: The disassembler follows the execution path. It is able to recognize subroutines and can include defined labels. You can interactively create own list files with comments from the disassembly.
  • NEW
    Call Graphs: Show a call graph of a subroutine. You can interactively change the calling depth.
  • NEW
    Flow Chart: Show a flowchart of a subroutine.
  • NEW
    Smart Disassembly: Output a smart disassembly into a window. Jumps and calls will be visualized with arrows. Here, as well, you can select different calling depths.
  • NEW
    Delta Search: An easy way to search for strings with encodings other than ASCII.
  • NEW
    'zsim' now supports custom memory maps. This offers more simulation opportunities. It is even possible to define custom slot ranges and custom banks. Also customized io based memory switching can be simulated.
  • CHANGED
    Memory views now support a search bar.
  • FIXED
    A lot of minor fixes and improvements.
